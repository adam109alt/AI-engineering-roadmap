# Input format

## What is (Input format)?

**Input format is a strctural format to give it to the LLM, Like making your prompt is mutch more detailed with specific structures to make the LLM understand you very good**

***And the ways to use this is:***
1. Task (e.g. Make to me a proffienal website)
2. Rules (e.g. Make hashtages in the code, use the html,css,js code in one html file)
3. Example (e.g. I want the website have the style of youtube)
4. Context (e.g. I'am a 21 years old and i want to build a proffissenal fitness website, and i love the light and the dark theme in the website)

And also it's good for using **XML tags** and the headers, and bold the text, and everything that make the AI focus on the thing

**Example**:

```
<task>Create a website</task>
<constraints>Use one file only, include comments</constraints>
<context>Fitness website for beginners</context>
```

This is a good example for what is input format is that we make the instructions very clear to the AI and the best thing to put it in the prompt is the **Bullet points**, And this is 100% Better than a messy code like: ***Build to me a proffiessnal website that is about fitness***
